# esercizio_gruppo_NMM
esercizio gruppo Nahira, Matteo, Michele
Suddivisione dei compiti:
Matteo si è occupato della creazione della repository e dei vari branch.

Nahira ha creato i pattern Singleton e Observer,
Matteo ha creato i pattern Factory e Decorator,
Michele si è occupato dei pattern Facade e Strategy.

In seguito, quando ognuno ha finito di fare i propri pattern abbiamo
deciso di completare l'esercizio singolarmente caricandolo sul proprio branch,
abbiamo scelto l'esercizio svolto di Nahira per la spiegazione,
nel proprio esercizio Nahira ha aggiunto l'extra per lo sconto del coupon rispettando gli altri requsiti presenti nella traccia.
